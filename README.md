# 铲什么铲

TFT 内容站重建项目，基于 Next.js App Router。

## 开发

```bash
npm install
npm run dev
```

本地访问 `http://localhost:3000`。

## 常用命令

```bash
npm run lint
npm run build
npm run start
```

## 当前范围

- 首页占位页
- NextAuth 登录
- MongoDB/Mongoose 用户模型
- `public/guides` 静态攻略素材

## 配置

复制 `.env.example` 为 `.env.local`，按需配置 MongoDB、NextAuth 和 GitHub OAuth。

敏感文件不要提交到 Git。
