# 观星者 霞

## 💡 概要

这是一套以4费攻速主C **霞** 为核心的阵容。

阵容自由位较多，最终上限很看能不能顺利补到5费英雄，所以更适合在8级小搜稳住后，血量和经济都还有余裕，再继续往9级运营时使用。

8级阶段通常要靠 **拉莫斯 / 纳尔 / 锐雯 / 库奇** 等单位临时搭棋盘，整体上手难度不低。由于 **努努和威朗普** 是热门卡，没升到2星前，主坦位置可以先让 **拉莫斯** 顶住。

## 🌄 适合开玩的条件

- 已经有 **鬼索的狂暴之刃**，并且准备围绕 **观星者** 体系运营。
- **观星者** 刷到的地形是 **秀山 / 野猪 / 蝰蛇 / 圣坛** 之一。
- 真正最值得深玩的情况是刷到 **秀山**。

只有在 **秀山** 时，才值得往高 **观星者** 方向走，所以通常也更容易顺着节奏升到 **Lv9**，并把8级阶段多出来的功能位补满。
