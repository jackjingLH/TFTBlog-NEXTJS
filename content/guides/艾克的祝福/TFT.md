# 艾克的祝福


**艾克的祝福**会在玩家生命值跌到 **30** 后，提供一个特殊的战利品法球。

## 🎁 奖励表

| 概率 | 奖励 |
| --- | --- |
| 22.5% | **20金币** |
| 22.5% | **3金币** + **2个随机基础装备** |
| 22.5% | **3个5费英雄** + **1个4费英雄** |
| 22.5% | **窃贼手套** |
| 10% | **英雄复制器** + **10金币** |
