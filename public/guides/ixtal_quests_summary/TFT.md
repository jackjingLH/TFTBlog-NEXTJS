# 以绪塔尔<br>各阶段任务推荐

## 📚 核心机制

### 基本原则
1. **任务类型**: 每次必定出现2个低风险任务 + 1个高风险任务
2. **任务刷新**: 同一任务可重复出现,但不会连续出现
3. **奖励计算**: 任务条件和奖励基于**接取时的阶段**,在后续阶段完成也按接取时奖励
4. **判定时机**: 任务判定在**战斗结束时**,在伤害结算后、回合结束前

### 收益合格线
- **阶段2**: 平均每回合 ≥25碎片
- **阶段3**: 平均每回合 ≥50碎片 ⭐
- **阶段4**: 平均每回合 ≥35碎片

## 🌟 阶段5特殊任务

### 💀 极限操作任务

**HP恰好为1** ⭐⭐⭐

`奖励` 700碎片!
`条件` HP在18-23之间

`对战完成法` (高风险)
1. 计算阶段5基础伤害10点
2. 敌方每存活1个单位+1伤害
3. HP=19时,对阵3个8级且全部存活 = 扣18血 = 剩1血
4. 出以绪塔尔(3),弱鸡站前排,一个不杀,完美受击

`野怪安全法` (推荐)
1. 阶段5野怪基础伤害10点
2. 先锋自身1点 = 共11伤害
3. HP=12时,野怪送死即可稳定达成!

`评价` 决定了就很帅,风险可控!

**HP低于5**

`奖励` 300碎片
`时机` HP在18-23时可接,输1场即完成
`风险` 控制不好容易暴毙

 **阶段3是一切的核心** - 收益最高的任务都在这里

 **优先完成速度** - 远比收益数字重要,1回合完成>2回合>3回合
