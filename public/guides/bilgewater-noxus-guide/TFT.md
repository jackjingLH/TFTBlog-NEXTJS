# 4套上分阵容推荐

## 🎯 版本概述

菲兹削弱后 16.4d版本，推荐四套阵容上分。

## ⭐ 第一套：比尔吉沃特九五阵容

![alt text](image-3.png)

- 二、三阶段：鬼索的狂暴之刃 + 肉装
- 连胜过四阶段 → 比尔吉沃特（7）+ 2星4费前排
- 先买装备和2星厄运小姐 + 1-2个经济强化符文
- 升至9级找5费主C
- 劣势局/多法流 → 加菲兹，装备后期分给吉格斯/希瓦娜

## ⭐ 第二套：诺克萨斯九五阵容

![alt text](image-4.png)

- 凑不齐比尔时优先此套
- 有免费D牌强化符文 → 解锁巴德，否则不强行
- 经济一般：8级 + 塞拉斯 + 千珏 → 上9级
- 物理系多：斯卡纳换塔里克，后期再换回塞拉斯

## ⭐ 第三套：约德尔人法师九五阵容

有小法师的约德尔人阵容依然强势，解锁不了直接转约德尔人九五。

## ⭐ 第四套：以绪塔尔岩宝阵容

**条件推荐**，开局有以绪塔尔羁绊时可尝试。

## 📊 版本梯队总结

上述四套强度断层明显，高分局主流阵容。
