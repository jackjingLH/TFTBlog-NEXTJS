# 如何玩好巴德

## 📖 概要

核心英雄**巴德**的主流玩法。

"第一个强化符文没拿到快速D牌相关的，还该不该玩巴德？"答案是YES。就算前期没有明确方向，利用好巴德的特性，照样能稳上分、冲第一。

## 🎯 优先强化符文

以下是能确保巴德解锁或与巴德配合超强的强化符文：

| 强化符文 | 运营要点 |
|---------|---------|
| 升星之运 | 场上面放同样的英雄，2-3回合D牌。能薅到超多免费D牌次数 |
| 耐心是一种美德 | 等2回合就白嫖D牌次数。想要的棋子用锁定商店解决 |
| DD街区 | 主打凯特琳的话2-1就能拿。不玩凯特琳就跟其他阵容一样2-3开D |
| 由二生D | 2-3开D，能找艾克、以绪塔尔什么的。<u>玩泰达米尔就别碰巴德</u>（强度会崩） |
| 英勇福袋 | 2-3开D，用复制器强行升星德莱文或蕾欧娜 |
| 新纪元 | 2-1直接开D。顺变升4级 |
| 棱彩门票 | 巴德专属顶级强化符文。根据门票给的免费D牌次数决定啥时候D。跟海克斯霸龙阵容也超配 |
| 沙海奥秘 | 开局炸了的救命稻草。能转黛安娜阵容 |

**升星之运小技巧**

这个符文每重叠一个上回合上过场的英雄就白嫖2次D牌，但如果盘面放2个同样的英雄，叠起来的时候能白嫖4次D牌。

## 🔄 4级D牌留牌标准

4级用巴德D牌时需要留的重点英雄清单：

**1费卡**:
- **嘉文四世**: 解锁塞拉斯或德玛西亚（配加里奥）。走德玛西亚记得连娑娜一起买
- **慎**: 前期顶级坦克。过渡艾欧尼亚或比尔吉沃特
- **俄洛伊**: 质量高的万能坦克
- **凯特琳**: 皮尔特沃夫起手卡

**2费卡**:
- **崔斯特**: 比尔吉沃特开羁绊必备
- **赛恩**: 叠上去能疯狂加速沃利贝尔解锁
- **蔚 / 妮蔻**: 以绪塔尔和护卫的优质拼图

**3费卡（阵容核心）**:
- **蕾欧娜**: 黛安娜阵容必拿
- **德莱文**: 能转95阵容或芸阿娜体系的万金油
- **薇恩**: 有"海妖之怒"的话贼猛。德玛西亚主C
- **阿狸**: AP流艾欧尼亚运营核心

## ⚡ 第二波发力点「3-5回合D牌」

这个赛季玩巴德最关键的就是<u>3-5回合D牌</u>。目标是完成以下3费2星：

- **2星蕾欧娜** → 转恕瑞玛·黛安娜阵容
- **2星薇恩** → 转7德玛西亚阵容
- **2星德莱文** → 转千珏95阵容

靠巴德免费D牌，7级抽到5费不是梦，特别是抽到阿兹尔、奥恩的话，名次直接起飞两名。

就算3-5回合D了，凭借巴德免费D牌加上德莱文、经济型艾欧尼亚、比尔吉沃特的黑市，经济回血速度快得很。

## 🎲 阵容支点与最终形态派生

把3-5回合D出来的2星单位当"支点"，转型最终阵容
