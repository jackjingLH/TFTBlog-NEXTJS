# 以绪 泰达米尔

## 🎯 阵容概述

这套阵容结合了中期战力超强的**泰达米尔**,以及通过长时间生存获取强力奖励的**以绪塔尔**特性,是一套非常独特且难度极高的争一阵容。

正如介绍所说,"冷门小众,玩法变态,难度拉满"。能玩的局面很有限,但一旦成型,爆发力在所有阵容里都是顶尖的。

## 📋 基础阵容

### 🚀 前期阵容（6级）

**核心单位:**
- **泰达米尔** / **艾希** / **瑟庄妮** → 3弗雷尔卓德
- **奇亚娜** → 裁决战士
- **妮蔻** → 护卫

## 🧠 阵容核心逻辑

这套阵容的强大之处,在于泰达米尔和以绪塔尔之间的完美互补。

**以绪塔尔特性:** <u>生存的回合越多,任务完成度越高</u>,积累的奖励(碎片)也就越多。这不是用来翻盘的特性,而是"当你优势时,能把优势转化为胜利的特性"。困境中它帮不了你。

**泰达米尔的作用:** 凭借强悍的战力守住中期血量,为以绪塔尔赚取完成任务所需的"时间"。

**协同效果:** 泰达米尔阵容通常只能拿3-4名,但加上以绪塔尔的高费奖励(岩宝等),就能弥补后期战力不足的问题,冲击第一。

## 🎮 运营要点

### 前置条件

只在<u>"已经领先"的局面</u>下才考虑玩这套:
- 拿到了**鬼索的狂暴之刃**等理想装备
- 前期泰达米尔或艾希已经升星

### 任务选择原则

选择以绪塔尔任务时要考虑三点:
1. **能不能完成?**
2. **几回合能完成?**
3. **符不符合当前节奏?** (正在D牌就选"刷新次数",正在升人口就选"购买经验值")

**碎片收益:** 550碎片左右性价比最高。

## 🎯 后期打法

### 转型升级流

**以绪塔尔泰达米尔**最终会转变成一套"升级流阵容"。

**争一关键:** <u>用任务奖励解锁岩宝</u>,然后在9级追岩宝2星。把泰达米尔积累的优势,通过岩宝的碾压级战力转化成胜利。

## 💭 心得体会

常规的快速D牌阵容,一旦追出3星就没有成长空间了,后期容易被"95阵容"(5费满地)翻盘。

但这套不一样。它用泰达米尔压制中期,然后用这段优势时间完成以绪塔尔"任务",准备好"岩宝"这个后期答案。

虽然对玩家水平和局势判断要求很高,但作为一套"不放弃现有优势,稳稳拿一"的思路,这套阵容确实非常合理。
